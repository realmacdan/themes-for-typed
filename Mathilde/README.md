# Mathilde - Typed Theme

This is an open source theme for Typed. For more information on building and modifying themes for Typed, please see the [themes SDK docs](https://www.typed.com/docs/themes/latest).

How to Install:
Compress the "theme" folder and upload the "theme.zip" file in the "Themes" tab under "Settings" on Typed.com.
